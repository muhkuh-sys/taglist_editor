Web server enable/disable

This parameter enables or disables the integrated web server.

Enable web server
  • unchecked: Disabled
    The web server is completely turned off (it is not initialized/started by the firmware) and therefore cannot be dynamically started (e.g. by the host application).
    Use case: the web server is not needed by the product the firmware is integrated in. Disabling the web server saves memory that can be used for other purposes.
  • checked: Enabled
    The firmware will start/initialize the web server with the configuration provided by the following taglist entries:
    Web Server (HTTP) Configuration
    Web Server (HTTPS) Configuration
