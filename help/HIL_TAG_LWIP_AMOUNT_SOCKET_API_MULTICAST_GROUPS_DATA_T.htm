Socket API IP Multicast Group configuration

With this tag, you can adjust the IP Multicast communication resources provided by the built-in IP stack integrated in the firmware.
The number of IP Multicast groups used by Socket API communication can be configured.

Number of IP Multicast groups for socket services Number of IP Multicast groups intended for Socket API usage
Default value: 0. Range: 0-65535
An individual firmware might define the maximum lower.
