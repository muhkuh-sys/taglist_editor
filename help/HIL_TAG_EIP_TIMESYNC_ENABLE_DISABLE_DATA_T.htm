CIP Sync support

This tag statically enables or disables support of the CIP TimeSync object and the underlying CIP Sync (IEEE1588) subsystem.

Enable CIP Sync
  • checked: The CIP Timesync object will be created during firmware startup.
    The host application can then dynamically register and unregister it through the packet API.
  • unchecked: The CIP Timesync object and the CIP Sync subsystems will be statically disabled in the firmware.
    This would allow for memory-constrained-platforms to assign more memory to other means.
